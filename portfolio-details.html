Electricity Consumption Analysis - Project Details

Project Overview

This project analyzes electricity consumption patterns across Indian states from January 2019 to December 2020. Using Tableau, I created interactive visualizations to explore consumption trends, regional variations, and the impact of COVID-19 lockdowns on energy usage.

Key Findings

  • Top Consumer: Maharashtra leads with 154,917 units consumed
  • Regional Leader: Western region (WR) shows highest overall consumption at 493,523 units
  • COVID Impact: 40% drop in consumption during lockdown (March-June 2020)
  • Seasonal Patterns: Peak consumption during summer months (May-June)
  • Bottom Consumers: Sikkim, Mizoram, and Nagaland show lowest consumption

Methodology

The analysis was performed using Tableau Public, with data cleaned and prepared from the Consumption.csv dataset. Visualizations include:

  • Geographic maps showing state-wise consumption
  • Bar charts for top/bottom consuming states
  • Time series analysis showing monthly and quarterly trends
  • Region-wise comparison dashboards
  • Storyboards highlighting key insights
