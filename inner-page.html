Electricity Consumption Analysis Details

About This Project

This project analyzes electricity consumption patterns across Indian states from January 2019 to December 2020. Using Tableau, I created interactive dashboards and storyboards to visualize consumption trends, regional variations, and the impact of COVID-19 lockdowns.

Key Findings

  • Top Consumer: Maharashtra leads with 154,917 units consumed
  • Regional Leader: Western region (WR) shows highest overall consumption at 493,523 units
  • COVID Impact: 40% drop in consumption during lockdown (March-June 2020)
  • Seasonal Patterns: Peak consumption during summer months (May-June)
  • Bottom Consumers: Sikkim, Mizoram, and Nagaland show lowest consumption

Data Source

The dataset contains daily electricity consumption records for all Indian states, including:

  • States and Union Territories across India
  • Regional classifications (NR, WR, SR, ER, NER)
  • Daily consumption values from Jan 2019 to Dec 2020
  • Geographic coordinates for mapping
